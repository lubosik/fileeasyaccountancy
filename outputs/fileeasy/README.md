## Analysis Scope

### Geographic Markets
- **Kent, UK**: Primary market including Maidstone, Canterbury, Medway, Ashford, Dartford, Gravesend, Tonbridge, Royal Tunbridge Wells, Sevenoaks, Sittingbourne, Gillingham
- **Dubai, UAE**: Secondary market including city and major free-zone contexts (DMCC/DIFC/JAFZA)

### Language
- English (both markets)

### Budget Cap
- Soft target: $50-$75 total
- Per niche group caps: ≤1,000 keyword rows per batch, ≤600 SERP rows total

## Tools Used
- **Perplexity MCP**: Query generation and intent labeling
- **DataForSEO MCP**: Keyword research, SERP analysis, competitor data
- **Firecrawl MCP**: Competitor website analysis

## Methodology
1. **Phase 0**: Setup & Estimator
2. **Phase 1**: Programmatic Query Generation
3. **Phase 2**: Expand & Score with DataForSEO
4. **Phase 3**: SERP & Competitor Audit
5. **Phase 4**: Opportunity Model & Shortlist
6. **Phase 5**: Content & Page Plan
7. **Phase 6**: Pricing, Funnels & CTA Strategy
8. **Phase 7**: Final Deliverable Pack

## Data Quality Standards
- Official location/language codes used throughout
- Intent labeling applied to all queries
- Aggressive deduplication maintained
- Clean artifacts with consistent formatting
- White-hat SEO practices only

## File Naming Convention
- All files use snake_case naming
- Geographic suffixes: `_kent` or `_dubai`
- CSV files for structured data
- JSON files for complex objects
- Markdown files for documentation
